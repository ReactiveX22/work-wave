# work-wave

A project for DBMS Course.

Employee Task Management, Visualizer.

Live Site: http://workwave.free.nf (not working currently)

## Project Features

**All Users:**

1. User Signup
2. User Login
3. User Logout
4. Profile Viewing
5. Update Password
6. Viewing Pending Role Requests

**Employees:**

1. Request Role
2. View Assigned Tasks and Completed Tasks
3. Upload Task File

**Admins:**

1. Role Approval
2. Supervisor Assignment

**Supervisors:**

1. Create Task
2. Task Assignment
3. View Task Data
4. Download Task
5. Mark Task as Completed
6. Delete Task
